# async static SendRegistrationToken(email, roleId) → {Promise.<any>}

Send the registration token via email. This checks if the user already has a token, (There shouldn't - if there is that means either invitation already sent or user registered) and then sends the token via email. When the user registers using the token, he/she will be given the role specified in role id. No user can register without a token.
Parameters:
Name Type Description
email string Email of the user
roleId string UUID of the role that should be assigned to user
